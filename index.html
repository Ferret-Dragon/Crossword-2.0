About

Greetings! This website began as a side project during a 12th internship. As students at CodeRVA, a group of friends and I enjoyed working together to solve crossword puzzles. Unfortunately, we had to put these away to make time to study--but did we really have to stop there?

I initially began assembling this project by looking at the code already made on this CodePen project by tuncay, then I realized that I had overlooked one very important thing. I wanted this project to allow users to insert their own notes into the crossword puzzle generator, but this also meant having to consider the fact that letters in crossword puzzle have to cross! Back to the drawing board...

So, after some internet surfing, I came across this amazing code by Mitchum. The algorithm itself is brilliant! It runs through a list of words, adds them to arrays, and generates the best possible combination! The caveat was that there was not space for user input. The answers are always displayed. Bummer. But if there was enough in common between the two algorthims, might there be a way to combine them? Yes!!! After about 2 weeks of frustration and accidentally deleting majors parts of the code on the way, I finally managed to combine the two programs using the format from the first puzzle, and the generator algorithm from the second.

Finally, the note algorithm itself was my own--with much assistance from the website W3schools, and emotional support from ChatGPT just to keep me sane. At the moment, I am still working through some glitches, but am very enthusiastic about the future of the project.
